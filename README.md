🌟 LUMI – AI Voice Assistant for the Visually Impaired
LUMI is an AI-powered voice assistant designed to empower visually impaired individuals by providing real-time guidance for everyday tasks. Using intelligent voice interaction and contextual AI, LUMI helps users navigate their surroundings, read text, operate appliances, and stay protected from payment fraud.
The goal of LUMI is to make the world more accessible, independent, and secure for people with visual impairments.
🚀 Features
🔊 Real-Time Voice Guidance
LUMI acts as a conversational voice agent that guides users through their environment and responds instantly to spoken commands.
🧭 Navigation Assistance
Provides step-by-step audio guidance to help users move safely through their surroundings.
📖 Text Recognition & Reading
Uses AI-powered OCR to read printed text, labels, and signs aloud.
🏠 Appliance Assistance
Helps users understand and operate household appliances by identifying buttons, controls, and instructions.
💳 Payment Fraud Prevention
Detects suspicious payment requests and warns users about potential fraud in both online and offline transactions.
🧠 How It Works
LUMI integrates multiple AI components to create a seamless assistive experience:
Speech Recognition for voice commands
Text-to-Speech for audio responses
Computer Vision for object and text detection
AI-based fraud detection for payment safety
Real-time voice agent interaction
🎯 Vision
Over 285 million people worldwide live with visual impairments. LUMI aims to bridge the accessibility gap by combining AI, voice technology, and computer vision to create a reliable digital companion for daily life.
